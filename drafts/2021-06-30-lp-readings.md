## Classics

- *A Machine-Oriented Logic Based on the Resolution Principle*, Robinson :
    arguably this is **the** foundational paper on logic programming. It
    presents unification. Fun aside : most modern statically-typed languages
    (Haskell, OCaml) using an algorithm built on top of this kind of unification
    to perform compile time type checking.
